##前言

Windows 环境下 有有两种Git工具， Git Bash 和Git Shell。这两种主要的区别是，Git Bash 是Linux风格的命令行工具，Git Shell 是Windows Power Shell 的风格，可以使用Windows 和Shell的命令来操作。我使用了 Git Bash。
